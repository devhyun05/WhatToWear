
## WhatToWearToday

I decided to create WhatToWearToday, a clothing recommendation website that provides real-time weather based suggestions. I decided to 
do this project because I often find myself in a dilemma when the temperature is around 10Â°C, unsure whether to opt for a hoodie or a jacket. 


## Tech Stack

**Frontend:** React, CSS3 

**Backend:** Node, Express

**Others:** Weather API, country API, unsplash API

## Pre-requisite

To run this project, you will need to add the following environment variables to your .env file


`COUNTRY_API_KEY=""`



## Run Locally


Install dependencies

```bash
  cd backend && npm install
  cd frontend && npm install
```

Start the server and run the program
```bash
  cd backend 
  node server.js
```

Start the frontend 

```bash
  cd frontend
  npm start
```
